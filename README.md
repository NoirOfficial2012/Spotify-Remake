# 🎧 Spotify Pro (Custom Web Music Player)

A fully functional, modern **Spotify-inspired web music player** built using **HTML, CSS, and JavaScript**. This project delivers a sleek UI, playlist management, favorites system, and an interactive audio player — all running directly in the browser with no backend required.

---

## 🚀 Features

### 🎵 Music Playback

* Play, pause, next, and previous controls
* Interactive progress bar (seek functionality)
* Volume control slider
* Real-time duration & current time display

### 📂 Playlists

* Create unlimited custom playlists
* Rename and delete playlists
* Add/remove songs from playlists
* Persistent storage using `localStorage`

### ❤️ Favorites System

* Add/remove songs to favorites
* Dedicated "Favorites" section

### 🔍 Smart Search

* Search songs by title or artist
* Works across all songs and playlists
* Clear search button for quick reset

### 🎨 UI & UX

* Dark / Light theme toggle
* Responsive and clean layout
* Hover actions for quick controls
* Smooth transitions and modern design

---

## 🧠 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Custom variables + themes)
* **JavaScript (Vanilla)** – Logic & interactivity
* **Font Awesome** – Icons
* **LocalStorage API** – Data persistence

---

## 🛠️ Customization

### ➕ Add Songs

Edit the `songs` array:

```js
{
  id: 6,
  title: "Song Name",
  artist: "Artist Name",
  audio: "audio-link.mp3",
  cover: "image-link.jpg"
}
```

### 🎨 Modify Theme

Edit CSS variables inside `:root` and `[data-theme="light"]`
